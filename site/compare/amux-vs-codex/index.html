amux vs OpenAI Codex

Local-first fleet vs. cloud-based agent. Different ecosystems, similar goals.

OpenAI Codex runs cloud-based coding agents in sandboxed containers. amux runs Claude Code agents locally on your machine with full environment access.

Key differences

FeatureamuxCodex
HostingLocal machineCloud sandbox
EnvironmentYour tools, configs, credentialsIsolated container
Parallel agentsUnlimited local sessionsCloud-based parallelism
Self-healingAuto-compact, restart, unstickCloud infrastructure
LLMClaude (Opus, Sonnet, Haiku)GPT-4o, o3
Open sourceYes (MIT)CLI open source, cloud proprietary
CostAPI tokens onlyOpenAI pricing

Get started with amux

Run dozens of Claude Code agents in parallel. Python 3 + tmux. Open source.

git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
amux register myproject --dir ~/Dev/myproject --yolo
amux start myproject
amux serve  # → https://localhost:8822
