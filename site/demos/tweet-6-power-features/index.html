Everything Else

The features you didn't know you needed.

Single Python file, ~12,000 lines. No build step, no Electron, no external services. Conversation forking, git-aware conflict detection, token tracking, built-in cron, file attachments (paste images or drag PDFs into the send bar), and a kanban board with iCal sync so agent deadlines show up in your calendar.

  • Conversation fork — clone any session's JSONL history onto a new branch
  • Git-aware conflict detection — warns when two agents share the same repo+branch
  • Token tracking from JSONL — daily spend per session with cache reads broken out
  • Built-in cron — schedule recurring commands, no crontab or systemd
  • File attachments — paste images or drag PDFs into the send bar
  • iCal sync — board deadlines appear in Google and Apple Calendar
