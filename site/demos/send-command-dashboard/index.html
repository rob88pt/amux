Send Commands from the Dashboard

Type prompts and commands directly to any Claude Code agent from the browser — no terminal, no tmux attachment needed.

How it works

  1. Click any session card to expand it
  2. Use the quick-action chips: /compact, /status, Ctrl-C, Enter
  3. Or type any prompt in the send bar and hit Send
  4. Commands queue offline and replay via Background Sync on reconnect

When to use this

  • Sending instructions to an agent without opening a terminal
  • Interrupting a stuck agent with Ctrl-C
  • Checking status across multiple sessions from your phone

Try amux

Run dozens of Claude Code agents in parallel. Python 3 + tmux. Open source.

git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
amux register myproject --dir ~/Dev/myproject --yolo
amux serve  # → https://localhost:8822
