Self-Healing Agents

Your agents don't need you as much as you think.

amux is self-healing — context window low? Auto-compacts. Thinking-block corruption? Restarts and replays. Stuck on a tool-approval prompt? Auto-responds. Your fleet runs truly unattended. The phone is for when you want to check in, not because you have to.

  • Context window low → auto-compacts before hitting the limit
  • Thinking-block corruption → detected, session restarted, last prompt replayed
  • Stuck on tool-approval prompt → auto-responds via the Esc-to-cancel marker
  • All silent — no notifications, no babysitting required
